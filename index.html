Xin Chào Đến Với CSS

CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả cách trình bày của tài liệu HTML. Nó cho phép bạn kiểm soát bố cục, màu sắc, phông chữ và nhiều thuộc tính khác của trang web.
